# 🛡️ BitBin

### A Secure, Modern, and Decentralized Cloud Storage Solution
---

## Key Features

* **End-to-End Encryption (E2EE):** Only you hold the keys. Your files are encrypted locally before they ever leave your device.
* **Intuitive File Management:** Effortlessly organize files and folders with a clean, responsive UI.
* **Smart Sync:** Instant, differential synchronization keeps all your devices updated without killing your bandwidth.
* **Effortless Collaboration:** Securely share files with links, setting granular access controls (view-only, edit, expiry dates).
* **Cross-Platform:** Native applications available for Web, Desktop (Windows, macOS, Linux), and Mobile (iOS, Android).

---

## Technology Stack

* **Frontend:** Vanilla Javascript
* **Backend:** Pure Node.js
* **Database:** MongoDB
